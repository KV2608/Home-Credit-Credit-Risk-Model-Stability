## Overview
This project focuses on building and validating robust credit risk prediction models using the Home Credit dataset (1M+ records). The objective was to assess model performance, stability, and generalization across time and segments.

## Dataset
- Home Credit Default Risk (Kaggle)
- Over 1 million loan application records

## Models Used
- LightGBM
- CatBoost
- XGBoost
- Ensemble learning

## Key Work
- Feature engineering and preprocessing
- Model training and hyperparameter tuning
- Stability and robustness analysis
- Ensemble modeling for improved performance

## Results
- Improved predictive performance by 18%
- Achieved Top 5% ranking (AUC 0.576) among 2,800+ teams in a global Kaggle competition

- Pandas, NumPy

- LightGBM, CatBoost, XGBoost
